How to launch the project

First, clone the repository, with the following command :

"Git clone git @ git.epitech.euquentin.tridon @ epitech.euDEV_dashboard_2019"

Once cloning has been carried out, the site consists of two parts.

- A front part, containing the visual aspect of the site, but also the implementation of our services.

- A back part, allowing the management of users, services and data.

It is necessary, using two respective terminals, to go to the 2 following directories with this command :
	- “cd dashboard-front” & “cd dashboard-back”
	- Once in these two directories, launch the following command "npm i" then launch "npm start"
	- A window should open in your default browser, on the following address httplocalhost3000, you should see the following page appear
